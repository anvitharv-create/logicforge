# LogicForge

LogicForge is an AI-powered C Programming Tutor that teaches programming logic before code.

Built with React, Node.js, Express, and Groq AI, LogicForge helps students learn C programming through AI-generated explanations, code reviews, and practice problem evaluation.

## Features

* AI-powered C programming tutor
* Logic-first teaching approach
* Code review and debugging assistance
* Practice problem evaluation
* Modern React frontend
* Express backend API
* Groq Llama 3.3 70B integration

## Tech Stack

### Frontend

* React 18
* Vite

### Backend

* Node.js
* Express

### AI

* Groq API
* Llama 3.3 70B Versatile

## Run Locally

### Backend

```bash
cd server
npm install
node server.js
```

### Frontend

```bash
cd client
npm install
npm run dev
```

Open:

http://localhost:5173
